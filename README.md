# 🎮 数独游戏 (Sudoku Game)

一个功能完整、界面美观的数独游戏Web应用，支持多种难度级别和游戏模式。

## 🚀 在线演示

🔗 **游戏链接**: [https://thegreedygiantpanda.github.io/Sudoku-Game/](https://thegreedygiantpanda.github.io/Sudoku-Game/)

## ✨ 功能特性

### 🎯 游戏核心
- **多种棋盘大小**: 4x4、6x6、9x9标准数独
- **五种难度级别**: 简单(500XP)、中等(1000XP)、困难(2500XP)、专家(4000XP)、地狱(5000XP)
- **智能提示系统**: 提供实时错误检测和提示功能
- **游戏统计**: 记录完成时间、错误次数、提示使用次数
- **自动保存**: 自动保存游戏进度，支持断点续玩

### 🏆 用户系统
- **等级系统**: 每5000XP升1级，展示玩家成长进度
- **经验值奖励**: 完成游戏获得对应难度的XP奖励
- **个人中心**: 查看游戏统计数据和个人信息
- **每日签到**: 每日登录获得100XP奖励

### 🎁 任务与成就
- **任务中心**: 完成特定任务获得额外奖励
- **成就徽章**: 解锁各种游戏成就
- **排行榜**: 查看全球玩家排名

### 🛠️ 工具系统
- **智能求解器**: 等级3以上解锁自动求解功能
- **错误检测**: 实时检测并标记错误输入
- **游戏重置**: 随时重置当前游戏
- **难度选择**: 灵活切换不同难度级别

## 🎨 界面设计

- **现代化UI**: 采用玻璃态设计和渐变色彩
- **响应式布局**: 完美适配桌面端和移动端
- **流畅动画**: 平滑的过渡效果和微交互
- **深色模式**: 支持系统深色模式自动切换

## 🛠️ 技术栈

- **前端框架**: 纯HTML5 + CSS3 + JavaScript (ES6+)
- **UI框架**: Tailwind CSS v2.2.19
- **图表库**: Chart.js v4.4.8
- **存储方案**: localStorage (本地存储)
- **构建工具**: 原生开发，无需构建

## 📦 安装与使用

### 本地运行
1. 克隆项目
```bash
git clone https://github.com/thegreedygiantpanda/Sudoku-Game.git
cd Sudoku-Game
```

2. 直接打开 `index.html` 文件即可运行游戏

### 部署到GitHub Pages
1. 确保项目已推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择分支和目录，点击保存
4. 几分钟后即可通过生成的链接访问

## 🎮 游戏指南

### 开始游戏
1. 点击"新游戏"按钮
2. 选择棋盘大小 (4x4 / 6x6 / 9x9)
3. 选择难度级别
4. 点击"开始游戏"

### 游戏规则
- 在空白格子中填入1-9的数字
- 每行、每列、每个小九宫格内不能有重复数字
- 错误输入会被高亮显示
- 可以使用提示功能获得帮助

### 快捷键
- **数字键1-9**: 快速输入数字
- **Delete/Backspace**: 删除当前格子内容
- **空格**: 标记候选数字

## 📊 游戏统计

游戏会自动记录以下数据：
- 总游戏局数
- 获胜次数和胜率
- 平均完成时间
- 最佳完成时间
- 总获得经验值
- 困难模式完成数
